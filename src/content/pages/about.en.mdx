## What this site is

Metro Typing is an independent guide and play page for a subway typing game. It helps visitors understand how the game works, start a round quickly, and read practical tips for improving Hangul typing rhythm with station names.

## Editorial approach

The articles and guides on this site are written for players who want a clear explanation before or after playing. We focus on subway line memory, typing accuracy, input settings, and ways to compare personal records without turning the page into an ad directory.

## Original game

The playable frame on the homepage loads the original game from [metrotyping.kr](https://metrotyping.kr/). The frame is provided for convenience. Copyright, availability, and the game experience itself follow the original service.

This website is **not** affiliated with a railway operator and does not publish official train schedules. Metro Typing is a typing game that uses maps and station names — not a transit information service.

For step-by-step modes and settings, see [How to play](/how-to-play). For coverage and line choice, see [Supported lines](/supported-lines).
